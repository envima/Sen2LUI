Model results

The figures show the model results depending on the overall dataset used for model training and testing (x axis) and the subdataset used for training (headline in plots). All plots together give an impression on how sensitive the model is regarding the available dataset for model training and testing (i.e.which years and exploratories).

For example, the first plot on the left shows the results if data from (i) 2017 and 2018, (ii) 2017, 2018, 2019, (iii) 2017 and 2019, and (iv) 2018 and 2019 was available for model training and testing. The performance is computed based on a left out dataset encompassing all data from 2017.

Hence, the very left entry on the x axis (ALL_2017_2018) indicates the model performance if the model is trained on data from 2018 and tested on data from 2017. The very right entry on the x axis (ALL_2018_2019) does not show any model performance because if only data from 2018 and 2019 is available, data from 2017 cannot be left out.
